  <h3> Projeto criado na aula da disciplina de programação Front-End, na Unicesumar, campus de Londrina. O professor [Leonardo Rocha](https://github.com/leonardossrocha) introduziu os conceitos de Framework, apresentando o Bootstrap e alguns exemplos disponíveis nesse framework, e git, realizando o passo-a-passo de configuração do ambiente de desenvolvimento.
   No desenvolvimento do Projeto foi implementado uma seção FAQ "Dúvidas Frequentes", contendo respostas para questionamentos de usuários, como recuperação de senha, confiabilidade e informações do site. O FAQ foi estruturado utilizinado HTML, CSS e JavaScript, permitindo a exibição dinâmica das respostas ao interagir com as perguntas.</h3>

   <h1 align="center"> 📚 O que aprendi: </h1>
    
    
   Neste projeto desenvolvido na disciplina de Programação Front-End, aprendi na prática como funciona a criação de uma página web.

Utilizei HTML para estruturar o conteúdo, organizando as páginas e fazendo a ligação entre elas. Com CSS, aprendi a estilizar os elementos e posicioná-los corretamente na tela, como no caso do botão de FAQ fixo no canto inferior direito.

Também utilizei JavaScript para adicionar interatividade, criando uma seção de FAQ onde as respostas aparecem ao clicar nas perguntas.

Além disso, aprendi a utilizar Bootstrap, entendendo como um framework pode facilitar e agilizar o desenvolvimento.

Outro aprendizado importante foi o uso de Git e GitHub, realizando o versionamento do projeto e publicando o repositório online.
